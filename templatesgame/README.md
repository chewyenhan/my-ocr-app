## 🌟 为什么选择这个游戏？

这不仅仅是一个游戏，更是一次**穿越时空的复习课**！

你是正在为历史考试头疼的初一学生吗？还是在寻找创新教学素材的历史老师？《马六甲：东方十字路口》紧扣独中初一历史课本内容，将枯燥的考点转化为充满挑战的生存抉择。

### 📚 深度契合独中历史课纲
游戏中埋设了大量课本核心考点，玩一遍胜过读三遍：
- **多元文化**：切身体验“84种语言”汇聚的国际港口。
- **商贸制度**：掌握金银锡币流通、货厂（官厂）管理与港务官（Syahbandar）的职能。
- **大国外交**：重温郑和下西洋与马六甲王朝的盟友关系。
- **宗教传播**：见证马六甲如何发展成为“小麦加”。

## 🎭 扮演你的历史角色

每个身份都有一段独特的故事，以及必须面对的终极 AI 挑战：

1.  **爪哇商人 (Merchant)**：在追求财富的同时，你是否能遵守港口法治，赢得港务官的尊重？
2.  **敦吡叻副官 (Adjutant)**：在暹罗大军的威胁下，你将如何协助敦吡叻防御？是否能说服郑和将军给予长期保护？
3.  **阿拉伯传教士 (Missionary)**：在多元文化的碰撞中，你如何推广信仰并协助苏丹打造东南亚的文明中心？

## 🧠 独创：AI 动态情绪交涉系统

本项目接入了 **Google Gemini 尖端人工智能**，为你带来前所未有的体验：
- **不是死板的选项**：你可以直接输入文字与 NPC 交流。
- **后果真实可见**：如果你在前面的剧情中表现不佳（数值过低），NPC 会对你表现出**极其厌恶与愤怒**的语气。
- **知识就是力量**：如果你能在对话中说出课本里的“关键词”（如：黄伞、官厂、84种语言），AI 会对你刮目相看，大幅提升成功率！

## 🛠️ 如何开始 (Quick Start)

1.  **准备秘钥**：前往 [Google AI Studio](https://aistudio.google.com/) 免费获取你的 API Key。
2.  **运行游戏**：下载本项目后，点击 `index.html` 即可在浏览器开启。
3.  **输入秘钥**：在首页输入你的 Key，点击“扬帆起航”，开启你的历史霸业！
