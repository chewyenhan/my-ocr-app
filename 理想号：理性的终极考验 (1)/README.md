### 🌟 专为马来西亚华文独中初二历史课设计

各位老师和同学们，欢迎登上“理想号”飞船！

当人类文明的火种面临最严峻的考验，你是否能凭借智慧与理性，在蛮荒异星上重建秩序？这款互动式历史思维游戏，将带你穿越时空，亲身经历启蒙思想家们所面对的社会难题，并通过你的每一个决策，重塑人类文明的未来！

---

### 📖 游戏特色：穿越启蒙，决策未来

「理想号：理性的终极考验」是一款沉浸式教育游戏，旨在帮助初二学生：

1.  **💡 亲身体验思想碰撞**：通过生动的叙事和选择题形式，模拟社会契约、权力制衡、自由市场等核心概念的形成过程。
2.  **🧠 理解启蒙思想精髓**：深入浅出地介绍卢梭、孟德斯鸠、亚当·斯密等思想家的核心主张，让抽象的理论变得具象化。
3.  **🗣️ 互动式学习，加深记忆**：内置语音朗读功能，辅助学生理解题目和哲学家的名言，提升学习体验。
4.  **🎯 培养批判性思维**：每一个选择都将影响文明走向，促使学生思考不同哲学思想的实践后果。

### 🎮 如何开始你的理性征途？

无需复杂安装，只需一个浏览器，即可开启知识探索之旅！

#### **方式一：在线体验 (推荐!)**

直接点击链接，立即开始游戏：
[**👉 立即体验《理想号：理性的终极考验》**](https://chewyenhan.github.io/Enlightment/)

#### **方式二：本地运行 (适合修改或开发)**

若需在本地电脑运行或修改游戏，请遵循以下步骤：

**前置要求:** 确保你的电脑已安装 `Node.js` (包含 `npm` 或 `yarn`)。

1.  **克隆仓库:**
    ```bash
    git clone https://github.com/chewyenhan/Enlightment.git
    cd Enlightenment
    ```
2.  **安装依赖:**
    ```bash
    npm install
    # 或者用 yarn install
    ```
3.  **启动开发服务器:**
    ```bash
    npm run dev
    ```
    然后，在浏览器中打开控制台显示的 `http://localhost:3000/` 或其他端口链接。

4.  **构建生产版本 (用于离线或部署):**
    ```bash
    npm run build
    ```
    完成后，你可以在 `dist` 文件夹中找到编译好的文件，直接用浏览器打开 `dist/index.html` 即可。
