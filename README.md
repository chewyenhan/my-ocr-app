## 🤔 为什么需要这个工具？

老师们经常需要把书本、试卷、板书拍照转成文字。传统方式：

> 📱 拍一张 → 提取文字 → 复制 → 再拍下一张 → 再提取 → 再复制……

**费时费力，重复劳动让人崩溃。**

## 🚀 快速开始

### 1. 打开网页

👉 **[https://chewyenhan.github.io/my-ocr-app/](https://chewyenhan.github.io/my-ocr-app/)**

### 2. 获取免费 Gemini API Key

本工具**完全免费**，但需要你自备 Google Gemini API Key（费用由 Google 收取，有每月免费额度）。

> 🔗 申请地址：[**Google AI Studio**](https://aistudio.google.com/apikey)

1. 点击上方链接，登录你的 Google 账号
2. 点击 **"Create API Key"**
3. 复制生成的 API Key
4. 回到 App，点击右上角 ⚙️ 设置图标，粘贴 API Key
5. 开始使用！

> 💡 Gemini 2.5 Flash 每月有**免费额度**，日常使用基本不花钱。

## 📱 使用教程

1. **设置 API Key** → 点击右上角齿轮图标，填入你的 Gemini API Key
2. **拍照/上传图片** → 点击「添加照片」上传或拖入图片
3. **点击图片提取** → 点击左侧队列中的图片，AI 自动提取文字
4. **或一键全部提取** → 点击「一键全部提取」批量处理所有图片
5. **导出结果** → 选择复制、下载 TXT 或打印为 PDF
